# JavaScript



## 기본 명령어

- alert("messege")
  
  - 알림창을 띄움
  
  - 확인을 요구
  
  - 받은 인자를 알창의 내용으로 사용

- prompt("messege", "answer")
  
  - 알림창을 띄움
  
  - 사용자의 입력을 요구
  
  - 첫번째 인자는 알림창의 메세지,
    
    두번째 인자는 디폴트 사용자의 입력으로 사용
    
    없을 경우

```javascript
const name = prompt("이름을 입력하세요.");
alert("환영합니다," + name + "님");
```
